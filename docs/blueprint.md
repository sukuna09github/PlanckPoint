# **App Name**: Planckpoint Website

## Core Features:

- Hero Section Slider: Autoplay slider on the homepage with smooth transitions, showcasing impactful visuals.
- Product Cards: Display the 6 products (AtomicAI™, BeyondQuadrants™, DisQuo™, ELWIS™, ProPRICING™, SkillOrbit™) in an easy-to-navigate layout with rounded corners and soft shadows.
- Insights Grid: Showcase research articles with clean, minimal cards featuring elevation and a subtle glow on hover.
- Global Presence Map/List: Display office locations via a stylized map or grid.
- Smooth Animations: Incorporate smooth fade-in and slide animations for section reveals and card load-ins.
- Contact Form: Allow users to send messages and inquiries.

## Style Guidelines:

- Primary color: Dark teal (#003E45) for a professional and modern feel.
- Secondary color: Teal accent (#007782) for highlights and interactive elements.
- Accent color: Bright cyan (#00A4C0) for CTAs and important visual cues.
- Background color: Off-white (#F4F7F7) to provide a clean and spacious canvas.
- Body and headline font: 'Helvetica Neue' (sans-serif) for a clean, modern aesthetic. Note: currently only Google Fonts are supported.
- Utilize a 12-column grid system for precise and responsive layout.
- Employ generous whitespace and consistent spacing throughout the design.
- Apply rounded corners (~12-16px) and subtle shadows for UI elements (cards, containers, buttons).
- Use smooth fade-in, slide-in animations for section reveals; hover-lift for cards.
